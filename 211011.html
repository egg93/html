HTML과 JS

3. HTML과 JS의 만남: script 태그

기본적으로 JS는 HTML 위에서 동작되도록 만들어진 언어

script 태그 안쪽에 JS가 오도록 약속되어 있음 -> 웹 브라우저는 script 안 쪽을 JS로 인식

이것은 HTML로 만든 문구임, 1+1은? 1+1

JS는 동적, HTML은 정적

4. HTML과 JS의 만남: 이벤트

onclick => HTML의 설명서에는
1. onclick 안의 속성은 반드시 JS가 와야 함.
2. onclick 속성의 속성값은 웹 브라우저가 기억하고 있다가 사용자가 클릭하면 JS의 문법에 따라 실행
즉, 어떤 사건 => 이벤트가 발생하면 어떠한 자바 스크립트가 실행되도록 하는 것 => onclick

검색이 제일 중요함

5. HTML과 JS의 만남(콘솔)

콘솔: 오른쪽 우클릭 => 검사 => console

개발자도구 모드에서 element 창에서 esc를 누르면 console 창이 함께 보여짐

console 창에서 위쪽 화살표 키를 누르면 이전에 실행된 명령이 다시 뜸
