    9. 줄바꿈

    인기 있는 태그 2가지: 줄바꿈 태그, 단락 태그 => 경쟁 관계

    줄바꿈 태그: <br>
    html new line tag

    예제
    To force<br>
    line breaks<br>
    in a text,<br>
    use the br<br>
    element.<br>

    설명
    Definition and Usage The <br> tag inserts a single line break.
    The <br> tag is useful for writing addresses or poems.
    The <br> tag is an empty tag which means that it has no end tag.

    단락 태그: <p>
    html paragraph tag (paragraph: 단락)

    예제

    <p>This is some text in a paragraph.</p>

    <p style="margin-top:40px">

    설명
    The <p> tag defines a paragraph.
    Browsers automatically add a single blank line before and after each <p> element.
    Tip: Use CSS to style paragraphs.

    의미론적으로는 단락 태그가 더 의미있을 수 있음.

    10. HTML이 중요한 이유

    기초는 중요. 응용으로 가는 과정으로도 중요하지만, 기초 자체만으로도 중요.

    일반인
    coding
    <span style='font-size:24px'>coding</span>
    VS
    지식인

    coding

    <h3>coding</h3>

    검색 엔진에 코딩을 검색을 했을 경우 일반인의 디자인적인 코딩보다 제목이 코딩인 지식인의 사이트의 힘을 더 들어줄 것.
    10년 후 같은 검색어로 지식인의 사이트는 초반 10페이지, 일반인의 사이트는 후반 1000페이지에 검색될 확률.
    비즈니스적인 측면에서도 의미론적인 코드 작성은 중요하다.

    디자인도 중요하지만 그 무엇보다 가장 중요한 첫 출발은 HTML 태그를 정확히 알고 의미에 맞게 사용하는 것.

    HTML 태그 => 비즈니스적 측면 + 휴머니즘적 측면, 매우 중요.
