24. 함수 예고

코드가 많아지면 그 코드를 정리정돈할 도구가 필요하
그 첫번째로 아주 간단하면서 강력한 도구가 바로 함수이다
그것보다 더 큰 두번째 도구는 객체.

night_day 버튼을 구현하는 input 태그가 1억개가 있다고 가장해볼 것.
문자 색의 변경이 필요할 때 1억개의 코드를 일일이 수정하는 것은 비효율적이고 정상적인 수정 작업 어려움 확률 up

onclick 안에 중복되는 코드 잘라내고 script 태그 안에 붙여 넣고 nightDayHandler()라는 함수로 구현, input 태그 onclick 안에 nightDayHandler() 함수 넣기.

<input type="button" value="this.night" onClick="
 nightDayHandler(this);
">

1. 반복되는 코드들이 한 줄로 줄어드는 것을 통해 웹페이지의 크기가 극단적으로 줄어들 수 있음
2. 동일한 함수가 쓰였다면 똑같은 로직을 갖고 있다라는 것을 확신 가능
3. 적절한 함수명을 통해 해당 함수가 어떠한 기능을 구현하고 있는 코드인지에 대해 파악 및 이해가 쉬움

26. 매개변수와 인자

입력:매개변수, parameter / 출력:인자, argumnent

<script>
function onePlusOne(){
 document.write(1+1);
};
onePlusOne();
</script>



=> onePlusOne이라는 함수는 언제나 똑같이 동작하는 함수.
함수에 입력값을 주면 그 입력값에 따라 다른 출력값을 내는 함수 구현 want

<script>
function sum(left, right){
 document.write(left+right+"<br>");
}
sum(2,3);
sum(3,4);
</script>



2, 3 / 3, 4 => argument
left, right => parameter
