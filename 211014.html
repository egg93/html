20. 배열

Array

Syntax
var refrigerator = ["banana", "cheeze"];
냉장고 = [바나나, 치즈]
배열은 [로 시작해서 ]로 끝난다.
그 안에 담기는 값들은 , 로 구분.

get
배열의 값이 담겨있는 변수명[값이 위치해 있는 순서]
document.write(refrigerator[0]);
document.write(refrigerator[1]);
값의 순서는 0번부터 시작

count
배열에 들어 있는 값이 몇개인 지 알고 싶을 때
추천 검색어: javascript array count
Array.length => document.write(refrigerator.length)

add
배열에 값을 더 추가하고 싶을 때
추천 검색어: javascript array add data
Array.prototype.push() => refrigerator.push("milk")
push 말고도 여러가지로 배열에 값을 추가하고 제거하는 등 다양한 메서드들이 존재하니 나중에 필요해지면 검색을 통해 알아가서 사용하면 된다.

21. 반복문

Loop
document.write('<li>1</li>');
document.write('<li>2</li>');
document.write('<li>3</li>');
document.write('<li>4</li>');
위와 같은 형식으로 코드해도 틀린 것은 아니나, 하나하나 코드를 짜기엔 번거롭고 손이 많이 간다. 만약 리스트의 개수가 1억개라면?

예제)
<script>
var i = 0;
while(i<3){
i = i + 1;
document.write('<li>'+i+'</li>');
}
</script>
while문 => 제어문 (cf. 반복문)
제어문의 실행 횟수를 제어하기 위한 변수 선언 필요.
var i = 보통 제어문에서 실행 횟수를 제어시키기 위한 변수명으로 관습적으로 i가 많이 쓰인다.
